## Description
A compact FM transmitter that picks up surrounding audio
and transmits it wirelessly over FM frequencies.

## How It Works
1. Microphone captures surrounding audio
2. FM Modulator varies carrier wave frequency
3. Antenna transmits the FM signal
4. FM Receiver captures and plays the audio

## Components Used
- Transistor (BC547)
- Inductor / Coil
- Capacitors & Resistors
- Condenser Microphone
- Antenna
- 9V Battery
